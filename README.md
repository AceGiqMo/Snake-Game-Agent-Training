# The Snake Game Agent Training via Particle Swarm Optimization (PSO)

This project offers non-standard training of the agent for **The Snake** game via nature inspired algorithm instead of traditional backpropagation. Each agent represents a neural network with **Input Layer (17 neurons)** + **Hidden Layer 1 (32 neurons)** + **Hidden Layer 2 (16 neurons)** + **Output Layer (4 neurons)**. The **Input Layer** accepts a state of environment and snake agent at the moment, and the **Output Layer** determines the next action of the snake (**UP**, **DOWN**, **LEFT**, **RIGHT**). In order to train the neural network, the randomly initialized **weights** and **biases** are converted into the coordinates of **high-dimensional point**. Thousands of points corresponding to specific neural networks configuration are scattered across the **high-dimensional space**. The point movement occurs according to the **Particle Swarm Optimization (PSO)** algorithm. At each epoch, thousands of game sessions are launched, and the result of each such session is converted into a **fitness** of corresponding point. Due to the **fast convergence** of the **PSO** algorithm, the agent can demonstrate visible progress in a few epochs. Moreover, the project offers the possibility to visualize the training (`training_visualizer.py`), by pre-projecting the high-dimensional points onto **2D-plane**, and save into `training_visualization/` folder as `.gif` files. Additionally, you can replay the game session of the best agent (`snake_replayer.py`)
